# Components

A collection of pre-built React components for Ethereum dApp development. These components provide a seamless way to display Ethereum addresses, balances, and handle user inputs.

# Installation

## Package Installation

Install Scaffold UI using your preferred package manager:

:::code-group

```bash [npm]
npm install @scaffold-ui/components @scaffold-ui/hooks
```

```bash [pnpm]
pnpm add @scaffold-ui/components @scaffold-ui/hooks
```

```bash [yarn]
yarn add @scaffold-ui/components @scaffold-ui/hooks
```

:::

## Peer Dependencies

Scaffold UI requires React as a peer dependency:

:::code-group

```bash [npm]
npm install viem wagmi @tanstack/react-query
```

```bash [pnpm]
pnpm add viem wagmi @tanstack/react-query
```

```bash [yarn]
yarn add viem wagmi @tanstack/react-query
```
