# Hooks

A collection of custom React hooks for Ethereum dApp development. These hooks provide powerful functionality for working with addresses, balances, and user inputs in your decentralized applications.

# Installation

## Package Installation

Install Scaffold UI using your preferred package manager:

:::code-group

```bash [npm]
npm install @scaffold-ui/hooks
```

```bash [pnpm]
pnpm add @scaffold-ui/hooks
```

```bash [yarn]
yarn add @scaffold-ui/hooks
```

:::

## Peer Dependencies
